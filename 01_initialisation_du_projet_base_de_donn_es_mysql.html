Module 1 : Initialisation du projet + Base de données MySQL

Dans ce premier module, nous allons poser les bases de notre API REST avec une structure professionnelle en architecture MVC. Nous utiliserons :

  • Node.js en mode ESM (ECMAScript Modules)
  • Express.js comme framework serveur
  • Sequelize avec mysql2/promise pour l’ORM
  • Une base de données MySQL

1. Initialiser le projet

Dans un nouveau dossier :

mkdir api-rest-node
cd api-rest-node
npm init -y

Modifiez le fichier package.json pour activer le support ESM :

{
  "type": "module",
  ...
}

2. Installer les dépendances de base

Installez les dépendances nécessaires :

npm install express sequelize mysql2 dotenv

3. Arborescence du projet

Organisez le projet avec une architecture MVC propre :

api-rest-node/
├── config/
│   └── database.js
├── controllers/
├── models/
├── routes/
├── middlewares/
├── app.js
└── .env

4. Configuration de la base de données

Créez un fichier .env à la racine :

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=motdepasse
DB_NAME=apirestdb

Fichier config/database.js

import { Sequelize } from "sequelize";
import dotenv from "dotenv";
dotenv.config();

const sequelize = new Sequelize(
  process.env.DB_NAME,
  process.env.DB_USER,
  process.env.DB_PASSWORD,
  {
    host: process.env.DB_HOST,
    dialect: "mysql",
    logging: false,
  }
);

export default sequelize;

Ce fichier initialise Sequelize avec vos variables d’environnement.

5. Point d’entrée : app.js

Créez un fichier app.js simple pour démarrer :

import express from "express";
import sequelize from "./config/database.js";

const app = express();
const PORT = process.env.PORT || 3000;

app.use(express.json());

// Test DB
try {
  await sequelize.authenticate();
  console.log("Connexion à la base de données réussie !");
} catch (error) {
  console.error("Erreur de connexion :", error);
}

app.get("/", (req, res) => {
  res.send("API opérationnelle !");
});

app.listen(PORT, () => {
  console.log(`Serveur démarré sur le port ${PORT}`);
});

6. Lancer le serveur

Ajoutez un script dans package.json :

"scripts": {
  "dev": "node app.js"
}

Et démarrez le serveur :

npm run dev

7. Résumé

À ce stade, vous avez :

  • Initialisé un projet Node.js en mode ESM
  • Installé les dépendances de base
  • Créé un serveur Express simple
  • Connecté Sequelize à une base de données MySQL

Dans le prochain module, nous allons créer le modèle User et mettre en place l'authentification avec JWT.
